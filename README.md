# rpc-ts-server

A TypeScript RPC server library.

## Installation

```bash
npm install rpc-ts-server
```

## Usage

```typescript
import { greet } from 'rpc-ts-server';

const greeting = greet('World'); // Returns: "Hello, World!"
```

## Development

1. Install dependencies:
```bash
npm install
```

2. Run tests:
```bash
npm test
```

3. Build the project:
```bash
npm run build
```

4. Lint the code:
```bash
npm run lint
```

5. Format the code:
```bash
npm run format
```
